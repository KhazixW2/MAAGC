## 安装问题

### Q1: 下载后无法解压

**问题描述**：
下载的压缩包无法解压或解压后文件不完整。

**解决方案**：

1. 使用 7-Zip 或 WinRAR 等专业解压软件
2. 确保下载完整（文件大小正确）
3. 尝试重新下载
4. 检查解压路径是否包含中文

### Q2: 运行时提示缺少文件

**问题描述**：
启动时提示缺少某些文件或文件夹。

**解决方案**：

1. 确保完整解压所有文件
2. 检查杀毒软件是否误删文件
3. 重新下载并解压
4. 将安装目录添加到杀毒软件白名单

### Q3: 应用程序无法正常启动

**问题描述**：
双击 maagc.exe 后无反应或弹窗报错。

**解决方案**：

1. 安装 [Visual C++ 运行库](https://aka.ms/vs/17/release/vc_redist.x64.exe)
2. 以管理员身份运行
3. 检查 Windows 版本（需要 Win10 或更高）
4. 查看日志文件 `debug/maa.log`

## 连接问题

### Q4: 无法连接到模拟器

**问题描述**：
连接模拟器时提示连接失败。

**解决方案**：

1. **检查 ADB 调试是否开启**
   - 进入模拟器设置
   - 确认已开启"ADB 调试"或"开发者模式"

2. **确认端口号正确**
   - 雷电 9：5555
   - 夜神：62001
   - MuMu：7555
   - 在模拟器设置中查看实际端口

3. **测试 ADB 连接**

   ```bash
   adb connect 127.0.0.1:端口号
   adb devices
   ```

4. **重启模拟器**
   - 完全关闭模拟器
   - 重新启动后再试

### Q5: 连接后很快断开

**问题描述**：
连接成功后很快自动断开连接。

**解决方案**：

1. 检查模拟器是否进入休眠状态
2. 关闭模拟器的节能模式
3. 增加模拟器性能分配
4. 检查 USB 调试授权是否过期

### Q6: PC 端无法识别游戏窗口

**问题描述**：
使用 Win32 控制器时找不到游戏窗口。

**解决方案**：

1. 确保游戏已启动并进入主界面
2. 游戏窗口不能最小化
3. 尝试手动调整窗口大小
4. 重启游戏和 MaaGC

## 运行问题

### Q7: 任务执行失败

**问题描述**：
任务开始后很快失败或报错。

**解决方案**：

1. **检查连接状态**
   - 确认设备连接正常
   - 重新连接设备

2. **检查分辨率**
   - 调整为 720*1280
   - 确保为 9:16 比例

3. **检查游戏状态**
   - 确保游戏在主界面
   - 关闭弹窗和公告

4. **查看日志**
   - 打开 `debug/maa.log`
   - 查找错误信息

### Q8: 执行速度很慢

**问题描述**：
任务执行速度明显慢于预期。

**解决方案**：

1. 降低模拟器分辨率
2. 减少模拟器性能分配
3. 关闭不必要的后台程序
4. 检查网络延迟（如使用远程连接）

### Q9: 任务卡在某个步骤

**问题描述**：
任务执行到某一步后不再继续。

**解决方案**：

1. 等待超时（默认 30 秒）
2. 手动干预后重新开始
3. 检查游戏是否有异常弹窗
4. 查看日志确定卡住的原因
5. 尝试降低执行速度

## 识别问题

### Q10: OCR 识别失败

**问题描述**：
提示"Failed to load det or rec"或"ocrer_ is null"。

**解决方案**：

1. **检查 OCR 资源文件**
   - 确认 `assets/resource/model/ocr/` 目录存在
   - 检查是否包含以下文件：
     - `det.onnx`
     - `rec.onnx`
     - `keys.txt`

2. **重新下载 OCR 资源**
   - 从官方渠道下载最新 OCR 包
   - 解压到正确目录

3. **检查文件权限**
   - 确保文件可读
   - 以管理员身份运行

### Q11: 识别不准确

**问题描述**：
经常识别错误或找不到目标。

**解决方案**：

1. **调整分辨率**
   - 使用 1280x720 标准分辨率
   - 避免使用非标准比例

2. **检查画面质量**
   - 提高模拟器渲染质量
   - 关闭抗锯齿等特效

3. **确保语言正确**
   - 游戏语言必须为简体中文
   - 繁体中文可能无法识别

4. **更新资源文件**
   - 下载最新版资源包
   - 替换旧版本

### Q12: 图像识别失败

**问题描述**：
找不到特定的图像或按钮。

**解决方案**：

1. 检查游戏 UI 是否有变化
2. 确认游戏版本是否最新
3. 更新 MaaGC 资源文件
4. 调整识别阈值（高级用户）

## 其他问题

### Q13: 日志文件在哪里

**问题描述**：
找不到日志文件。

**解决方案**：
日志文件位置：

- **Windows**: `安装目录/debug/maa.log`
- 直接在安装文件夹中查找 `debug` 文件夹

### Q14: 如何提交问题反馈

**问题描述**：
遇到问题不知道如何反馈。

**解决方案**：

1. **收集信息**
   - 日志文件 `debug/maa.log`
   - 截图或录屏
   - 问题复现步骤

2. **提交 Issue**
   - 访问 [GitHub Issues](https://github.com/KhazixW2/MAAGC/issues)
   - 点击"New issue"
   - 选择问题模板
   - 填写详细信息

3. **提供必要信息**
   - MaaGC 版本号
   - 操作系统版本
   - 模拟器类型和版本
   - 游戏版本

### Q15: 是否安全

**问题描述**：
担心使用自动化脚本的安全性。

**解答**：

1. **代码开源**
   - 所有代码在 GitHub 公开
   - 可审查无恶意代码

2. **本地运行**
   - 所有操作在本地完成
   - 不上传账号信息

3. **遵守协议**
   - 请遵守游戏用户协议
   - 合理使用，不要作弊

4. **风险自担**
   - 自动化存在一定风险
   - 请自行评估后使用

### Q16: 会被封号吗

**问题描述**：
担心使用辅助工具导致封号。

**解答**：

1. **风险提示**
   - 任何自动化工具都有风险
   - 无法保证 100% 安全

2. **降低风险建议**
   - 不要 24 小时挂机
   - 模拟正常玩家行为
   - 避免在活动期间大量使用
   - 准备小号测试

3. **责任声明**
   - 使用后果由用户自行承担
   - 开发者不对此负责
